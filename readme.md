The Concept
=======

This repo outlines the concept of Yip, the software and the public platform.

## Generation

Markdown to HTML generation is done via daux.io.

Requirements:

 * PHP5
 * Node.js
 * `brew install httrack`
 * grunt ([globally installed](http://gruntjs.com/getting-started))

```sh
> node ./generate.js
```
